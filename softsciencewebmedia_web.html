SoftScience Web Media

As a web designer at SoftScience Group, I designed a website for SoftScience Web Media. SoftScience Web Media is a division of SoftScience Group, Inc. that specializes in new media development. The website uses bold colors, CSS3 animations and parallax scrolling to achieve the look and feel. The 'resources' section of the site is a mini-site, that functions as a place for other web developers and professionals in the industry to go to find helpful resources and instruction. The website's main goal is to establish credibility by taking visitors through the web development process. The 'Explore' section briefly exaplins the components of web development as well as how to establish a healthy web presence, through avenues such as social media, strategic content, SEO and analytics. This was created to exhibit SoftScience Group's knowledge and expertise in web development.

SoftScience Web Media is a division of SoftScience Group, Inc., a 25 year old software development firm based in Kansas City. SoftScience Web Media specializes in new media development.
