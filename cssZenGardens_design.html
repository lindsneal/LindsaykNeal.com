CSS Zen Garden

CSS Zen Garden is a website that showcases pure CSS design. The website contains different designs of the same html page styled by various people, using only CSS. The site displays the submissions to highlight the creative things you can do with CSS. In my advanced CSS course, we came up with our own version of the CSS Zen Garden html page using CSS3 techniques to manipulate the page.
