Petroleum Science Corp

As a web designer at SoftScience Group, I was the lead designer on a project for the software company, Petroleum Science Corporation. This project allowed me to gain experience using the CSS framework, Bootstrap. The website design was focused on photography. Large image files were used throughout the site, which meant that optimization was a key focus for the project to better the website performance.

Petroleum Science Corporation develops an assortment of different software products specialized for the petroleum industry.
